What is Docker

Docker is one of the cloud computing tool used to run the software package called “Containers”. Containers allow a developer to package up an application with all of the parts it needs, such as libraries and other dependencies, and ship it all out as one package. Containers use the shared operating system and they are much more efficient than hypervisors in system resource

What is Nginx

Nginx is a web sever, which behaves as a reverse proxy server, load balnacer and HTTP Cache. It is an open source software. Almost of the world's websites. It provides more predictable performance under high load because of the asynchronous event-driven architecture it uses.

Resouces:- Wikipedia, Nginx.com.

What is VM (virtual machine)

A virtual machine (VM) is an emulation of a computer system just like creating a computer within a computer. It runs like any other program, giving the end user the same experience on a virtual machine as they would have on the host operating system itself. Virtual machines are based on computer architectures and provide functionality of a physical computer. Their implementations may involve specialized hardware, software, or a combination

Who is Cloud Computing Service Provider - Digitalocean

DigitalOcean, Inc. is an American cloud infrastructure provider headquartered in New York City with data centers worldwide.

It acts as a virtual web based host and also offer the block & object based storage.
